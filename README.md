# 🎓 Repositório de Disciplinas e Planos de Aula

Este repositório centraliza os cronogramas, conteúdos programáticos, diagramas arquiteturais e materiais práticos das disciplinas sob minha regência. O foco das aulas está na aplicação de metodologias ativas, simulações em laboratório e prática intensa de código.

---

## 📚 Sumário das Disciplinas

### 🌐 1. Arquitetura de IoT
Aborda o desenvolvimento de ecossistemas de Internet das Coisas ponta a ponta, desde a coleta física até o tratamento em nuvem ou gateways locais.
*   **Hardware Principal:** Arduino (Uno, Mega, Nano)
*   **Linguagens:** C++ (Firmware/Embarcados) e Python (Scripts, processamento e leitura serial com `pySerial`)
*   **Foco Prático:** Leitura de sensores (DHT22, LDR), atuadores (Relés), comunicação UART e visualização de dados.

### 📋 2. Levantamento de Requisitos
Capacita os alunos nas etapas de engenharia de software para mapear as reais necessidades de negócio e transformá-las em documentação técnica sem ambiguidades.
*   **Técnicas de Elicitação:** Condução de **Entrevistas** estruturadas e dinâmicas de **Brainstorming**.
*   **Classificação:** Mapeamento rigoroso de **Requisitos Funcionais (RF)** e **Requisitos Não Funcionais (RNF)**.
*   **Modelagem e Entrega:** Criação de Diagramas de Casos de Uso (UML) e estruturação de **Relatórios Técnicos (SRS - IEEE 830)**.

### 💻 3. Lógica de Programação Multilinguagem
Ensina os fundamentos universais do pensamento algorítmico, ramificando a lógica em três ecossistemas distintos do mercado tecnológico.
*   **Lógica Web:** Estruturação visual com **HTML5** e comportamento dinâmico com **JavaScript** básico.
*   **Lógica de Scripts:** Sintaxe limpa, tratamento de dados, condicionais e repetições usando **Python**.
*   **Lógica de Hardware:** Automação e controle físico de microcontroladores rodando **MicroPython**.

### 🐧 4. Sistemas Operacionais
Apresenta os conceitos teóricos do núcleo dos sistemas e a administração prática de servidores e estações de trabalho via linha de comando.
*   **Sistemas Alvo:** Microsoft **Windows** e Linux (**Debian** GNU/Linux).
*   **Prática em Linha de Comando:** Operação avançada via **Prompt de Comando (CMD)** e **Terminal Bash**.
*   **Conceitos de Infraestrutura:** Gerenciamento de processos, permissões de arquivos (`chmod`/`chown`), sistemas de arquivos (NTFS vs EXT4) e gerenciamento de memória.

## 📚 Cronograma e Conteúdo das Aulas

### 📝 Módulo 1: Fundamentos de Hardware e C++ no Arduino
*   **Aula 01:** Introdução à Arquitetura do Arduino (Microcontrolador ATmega, pinos GPIO, PWM).
*   **Aula 02:** Sintaxe do C++ para Arduino: Estrutura básica (`setup` e `loop`), variáveis e tipos de dados.
*   **Aula 03:** Controle de fluxo em C++ (estruturas condicionais e laços de repetição).
*   **Aula 04:** Manipulação de entradas e saídas digitais (Leitura de botões e controle de LEDs).
*   **Aula 05:** Manipulação de entradas analógicas (Leitura de sensores como LDR e potenciômetro).

### 📝 Módulo 2: Sensores Avançados e Comunicação Serial
*   **Aula 06:** Integração de sensores complexos em C++ (Uso de bibliotecas para DHT11/DHT22 e Ultrassônico).
*   **Aula 07:** Protocolos de comunicação de baixo nível: Introdução ao I2C e SPI.
*   **Aula 08:** Comunicação Serial (UART): Enviando dados do Arduino para o computador.

### 📝 Módulo 3: Integração com Python e Processamento de Dados
*   **Aula 09:** Introdução ao Python para IoT e configuração do ambiente (Instalação e gerenciadores de pacotes).
*   **Aula 10:** Biblioteca `pySerial`: Lendo os dados do Arduino em tempo real via script Python.
*   **Aula 11:** Tratamento e limpeza de strings no Python vindas do hardware.
*   **Aula 12:** Armazenamento de dados: Salvando leituras do Arduino em arquivos CSV com Python.

### 📝 Módulo 4: Visualização e Arquitetura de Aplicação
*   **Aula 13:** Visualização de dados de sensores com Python (Gráficos em tempo real com `matplotlib`).
*   **Aula 14:** Controle reverso: Python enviando comandos para o Arduino atuar no ambiente (Ligar relés/motores).
*   **Aula 15:** Apresentação de Projetos Finais: Sistema IoT ponta a ponta (Arduino C++ ➔ Python).

## 🛠️ Ferramentas e Requisitos Práticos

### Hardware Recomendado
*   Placa Arduino (Uno, Mega ou Nano)
*   Cabo USB de dados
*   Sensores: DHT22 (Temperatura/Umidade), LDR (Luz)
*   Atuadores: LEDs, Buzzer, Módulo Relé 5V

### Software e Bibliotecas
*   **Para Hardware (C++):** Arduino IDE ou VS Code (PlatformIO)
*   **Para Software (Python):** Python 3.x, bibliotecas `pyserial` e `matplotlib`
